# AIShortHub

AIShortHub is a short-drama platform frontend shell upgraded from a single-file prototype.

## Current frontend scope

- Multi-page route structure (home, browse, series detail, watch, submit, creator dashboard, pricing, admin, login, signup, forgot password).
- Componentized UI and separated mock datasets.
- Future integration boundaries for auth, playback/CDN source, distribution analytics, and billing APIs.
- Mobile-first responsive layout for browse/watch/submit/dashboard workflows.

## Run

```bash
npm install
npm run dev
npm run build
```
